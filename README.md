# 🚀 My Portfolio

Welcome to my personal portfolio website! This project showcases my skills, projects, and experience in an interactive and visually appealing way. Built using **React**, this site serves as a central hub for everything about me.  

## 🌟 Features
- **Landing Page** – A welcoming introduction with eye-catching design.
- **About Me** – My background, skills, and interests.
- **Projects** – A showcase of my best projects with descriptions and links.
- **Contact** – Ways to reach out to me.

## 🛠️ Tech Stack
- **Frontend**: React, Tailwind CSS
- **Hosting**: Glitch
